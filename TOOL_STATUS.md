## Architecture Notes

- **45 tools** total defined in `client/src/lib/tools.ts`
- **22 dedicated page components** in `client/src/pages/`
- **23 tools** use the generic `AIToolChat` component (still fully functional with tool-specific system prompts)
- **35 items** in sidebar nav (not all tools — some are accessible via Dashboard stage cards and Cmd+K search)
- All tools accessible via the Cmd+K search bar which searches `allTools`
- System prompts are sent client-side to `/api/chat` endpoint
- Chat history persisted per-tool in `chat_messages` table
